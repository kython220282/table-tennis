# Table Tennis Game

## Overview
The Table Tennis Game is a single-player web application where the player competes against a computer-controlled opponent. This game provides an engaging and interactive experience with simple controls.

## Features
- Single-player mode against a computer opponent
- Real-time scoring system
- Smooth paddle and ball movement
- Option to restart the game

## How to Use
1. Open the `main.html` file in any modern web browser.
2. Use the keyboard to control the paddle and hit the ball.
3. The game keeps track of the score in real-time.
4. Click the restart button to reset the game and play again.
